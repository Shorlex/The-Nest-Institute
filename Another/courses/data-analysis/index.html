Concepts




Data Analysis with Excel

Explore everything from fundamental formulae to sophisticated functions and powerful features that facilitate efficient data analysis and accurate outcomes in Microsoft Excel.



Sorting, Filtering and Formatting

Data sorting is any process that involves arranging the data into some meaningful order to make it easier to understand, analyze or visualize. When working with research data, sorting is a common method used for visualizing data in a form that makes it easier to comprehend the story the data is telling.



Charts and Pivot Tables

A Pie Chart is a sort of graph that shows data in the form of a circular graph. The graph's parts are proportionate to the fraction of the total in each group. In other words, each slice of the pie is proportional to the size of that category in the overall group. The full "pie" represents one hundred percent of the total, whereas the pie "slices" represent segments of the whole.

Unit 2

Introduction To SQL

A spreadsheet is a piece of software that allows you to quickly conduct mathematical calculations on statistical data, such as totalling numerous columns of numbers or calculating percentages and averages. And if any of the raw data you enter into your spreadsheet change - for example, if you acquire final values to replace preliminary ones - the spreadsheet will update all the calculations you've done based on the new numbers. In this unit, our focus will be on Excel.

Concepts



Logical and comparison Operators

In a structured query language (SQL) statement, logical operators are used to describe conditions. They are also used as conjunctions in statements with several circumstances.



Aggregate Functions and Expressions

An aggregate expression applies an aggregate function to the rows or groups of data that a query selects. An aggregate expression can only exist in a SELECT statement's select list or HAVING clause. It is ineffective in other clauses, such as WHERE, since such clauses are assessed before aggregate results are produced.



JOINS and UNIONS

The SQL JOIN clause is used to join records from two or more databases tables. A JOIN is a method of joining fields from two tables by using values that are shared by both. SQL UNION combines the output of two or more SELECT statements.
